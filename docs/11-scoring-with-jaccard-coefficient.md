## 重點摘要 (Key Summary)

- **Jaccard 係數定義：** Jaccard 係數等於兩個集合交集的大小除以聯集的大小，其值介於 0 到 1 之間。集合與自身的 Jaccard 係數為 1，不相交集合的 Jaccard 係數為 0。
- **應用於文件檢索：** 將查詢和文件視為詞語集合，計算兩者的 Jaccard 係數作為匹配分數，重疊詞語越多、總詞語數越少，分數越高。
- **較短文件的優勢：** 在其他條件相同的情況下，較短的文件會獲得較高的 Jaccard 分數，這是資訊檢索模型中常見的概念。
- **Jaccard 係數的缺點：**
  - 不考慮詞頻（忽略詞語在文件中出現的次數）。
  - 不區分稀有詞項和常見詞項（稀有詞項在評估查詢時更具資訊量）。
  - 以聯集進行正規化的方式不夠理想，餘弦相似度使用聯集的平方根作為分母是更好的長度正規化方式。
- **引出後續主題：** 好的檢索模型需要處理詞頻、詞語稀有程度和文件長度正規化等問題。
