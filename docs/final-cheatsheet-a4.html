期末考 A4 速記表 (Ranked Retrieval / TF-IDF)

10 題關鍵字（缺即扣分）

#必寫關鍵字
Q1Boolean → Feast or Famine（AND少/OR多）→ Ranked → Scoring (0~1) + 只看前面
Q2⚠️ 5 面向（必齊）：匹配/結果/查詢/F-F/對象。Boolean: Exact Match / Set / Boolean Query / 專家 ↔ Ranked: Scoring / Ranked List / Free Text Queries / 一般人
Q3|A∩B| / |A∪B| + 自己編範例算 + 缺 TF + 缺稀有度
Q4Bag of Words = 不管詞序、只管詞頻。Binary→Count Vector。例: "John is faster than Mary" vs "Mary is faster than John" → 向量同、意思反
Q5w = 1 + log₁₀(tf) if tf>0；0 if tf=0。為何用 log？Sublinear Scaling（次線性縮放，避免 log(0)=-∞）。tf=10→w=2、tf=1000→w=4
Q6IDF(t) = log₁₀(N / df_t) ⚠️ 是 df 不是 tf！N=10⁶: df=1→6, df=100→4, df=10⁶→0
Q7TF/DF/CF：TF=單一文件次數、DF=文件數、CF=集合總次數。⭐ IDF 對單一詞項查詢無影響，只對 multi-term query 有用。try (DF高) vs insurance (DF低) → DF 反映鑑別力
Q8⭐ TF-IDF = (1 + log₁₀ tft,d) × log₁₀(N / df_t)。連字號不是減號 × 不是 -。詞頻 × 稀有度。Manning：「只能學一個 IR 就學這個」
Q9Score(q,d) = Σ TF-IDF(t,d)，for t ∈ q ∩ d（只算共有詞！）。範例需有 Query + Doc + 共有詞 + 加總
Q10每階段都要寫公式！Jaccard: |A∩B|/|A∪B| → TF: 1+log₁₀ tf → IDF: log₁₀(N/df) → TF-IDF: (1+log tf)×log(N/df)

核心公式（必背）

TF:     w = 1 + log₁₀(tf)  if tf>0
                  = 0          if tf=0

IDF:    IDF(t) = log₁₀(N / df_t)
                          ↑↑↑
                       是 df 不是 tf！

TF-IDF: (1+log₁₀ tf) × log₁₀(N/df) ⭐
        └─ TF ─┘    └── IDF ──┘
            注意 × 不是 -

Score:  Score(q,d) = Σ TF-IDF(t,d)
                    for t ∈ q∩d
                    ↑↑↑ 共有詞才算！

數字記憶表

tf=1     → w=1
tf=2     → w=1.3
tf=10    → w=2
tf=1000  → w=4
（變 10 倍、w 只多 1）

(N=10⁶)
df=1     → IDF=6
df=100   → IDF=4
df=10K   → IDF=2
df=10⁶   → IDF=0 (log(1)=0)
（df 越大、IDF 越小）
⚠️ 五大死穴（每題都要小心）
  1. Q6 / Q8 公式用 df_t 不是 tf_t（IDF = Inverse Document Frequency）
  2. Q8 TF-IDF 是相乘 ×，不是相減 -（中間是連字號）
  3. Q7 IDF 對單一詞項查詢無影響，只對 multi-term query 有用
  4. Q4 必寫 John/Mary 例子（向量同、意思反）
  5. Q2 五個面向缺一不可，最容易漏「匹配方式」
📝 6/12 練習心得（自己最容易錯的地方）
  1. Q2：「五個面向」念三次→ 匹配/結果/查詢/F-F/對象
  2. Q5：「Sublinear Scaling」次線性縮放，sub=低於、linear=線性
  3. Q6：IDF 念三次 → 用 df（Document Frequency）
  4. Q8：寫公式時看清楚 df_t（不是 d_t、tf_t、df_d,t）
  5. Q9：範例一定要有「Query + Doc + 共有詞 + 加總」
  6. Q10：每階段都要寫公式（不只講概念）

整體演進邏輯（Q10 用）

Boolean (Exact Match, 無排名)
   ↓ 解決 Feast/Famine
Ranked Retrieval (Scoring 0~1)
   ↓ 開始評分
Jaccard |A∩B|/|A∪B|          ← 缺 TF + 缺 IDF
   ↓ 加詞頻
TF: w = 1 + log₁₀(tf)         ← Sublinear Scaling
   ↓ 加稀有度
IDF: log₁₀(N/df)              ← 只對 multi-term query 有用
   ↓ 合體 ×
TF-IDF: (1+log₁₀ tf) × log₁₀(N/df) ⭐ 經典加權
   = 詞頻 × 稀有度
