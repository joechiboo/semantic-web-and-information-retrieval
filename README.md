# Semantic Web and Information Retrieval

語意網路與資訊檢索 - 課程筆記

## 課程內容

本專案整理自 YouTube 公開講座影片，涵蓋以下主題：

- **Information Retrieval (資訊檢索)**：來自 Stanford 大學 Christopher Manning 教授的 [Introduction to Information Retrieval](https://nlp.stanford.edu/IR-book/) 系列課程
- **Semantic Networks (語意網路)**：來自多位講師的公開教學影片
